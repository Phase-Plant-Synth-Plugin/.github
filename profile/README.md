## 🚀 Overview
Phase Plant for Windows is a modular synthesizer from Kilohearts built around a simple idea: instead of a fixed layout, you assemble the instrument yourself. The Phase Plant synth stacks generators such as wavetable, analog, sampler, and noise, then routes them through filters and a lane of drag-and-drop effects called snapins. Because the Phase Plant VST loads inside your host DAW, producers and sound designers can start from a blank patch and grow it into a bass, lead, pad, or evolving texture without leaving their session. The flexible modulation system ties everything together, so any control can move over time.

## ✨ Features
- **Modular Generators** — Combine wavetable, analog, sampler, and noise generators in a single patch to build the exact starting tone you want.
- **Drag-and-Drop Snapins** — Place modular effects like distortion, delay, reverb, and filters directly in the signal path and reorder them freely.
- **Flexible Modulation** — Route envelopes, LFOs, and other modulators to almost any parameter to add motion and expression.
- **Visual Signal Flow** — See your generators, filters, and effect lanes laid out clearly so complex patches stay easy to follow.
- **DAW Integration** — Load the Phase Plant plugin as a VST in your host, automate parameters, and recall every patch with your project.

## 🎯 Getting Started
1. Open your DAW and create a new instrument track, then insert Phase Plant as a VST.
2. Add a generator such as a wavetable or analog oscillator to hear a starting tone.
3. Drop a filter after your generators and add a few snapin effects to the effect lane.
4. Assign an envelope or LFO to a parameter to bring movement into the patch.
5. Save your creation as a preset so you can recall and refine it later.

## ❓ FAQ
**Is Phase Plant free?**
Phase Plant is a paid plugin from Kilohearts, so there is no free full version. Searches for a Phase Plant VST free download usually point to a trial or demo you can use to evaluate the synth before buying, and this project does not provide any way around licensing.

**How does Phase Plant compare to Serum?**
Both are well-respected sound-design tools with different approaches. Phase Plant is a modular synth where you stack generators and route them through filters and snapin effects, while Serum is a wavetable synth known for its visual wavetable editing. Comparisons of Phase Plant vs Serum, and even Phase Plant vs Serum 2, usually come down to whether you prefer a build-it-yourself modular setup or a focused wavetable workflow, and many producers use both.

**Which versions of Windows are supported?**
Phase Plant for Windows runs on current 64-bit versions of Windows, including Windows 10 and Windows 11. Check the installer notes for the exact supported releases.
